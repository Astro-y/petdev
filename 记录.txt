可以，已经给你加了一个更省事的本地命令：npm run add:pet。

以后新增宠物只要准备一个包含 pet.json 和 spritesheet.webp 的文件夹，然后运行：

npm run add:pet -- --source "C:\path\to\pet-folder" --tag v1.0.1 --zh-name "中文名"
它会自动做三件事：

复制到 public\pets\<pet-id>\
自动追加 packages\petdev\src\catalog.js
自动生成 release\<pet-id>.zip
然后你只剩三步：

把 release\<pet-id>.zip 上传到 GitHub Release 对应 tag，比如 v1.0.1
把 packages\petdev\package.json 版本号加一下，比如 0.1.1
发布 npm：
npm publish --workspace packages/petdev --access public
